Building jetty-nosql-memcached
--------------

You must first checkout clone of jetty.project.

  $ git clone git://github.com/eclipse/jetty.project.git
  $ cd jetty.project
  $ git checkout jetty-8.0.2.v20111006
  $ mvn -Dmaven.test.skip=true clean install

Then you can build jetty-nosql-memcached.

  $ git clone git://github.com/yyuu/jetty-nosql-memcached-parent.git
  $ cd jetty-nosql-memcached-parent
  $ git submodule init
  $ git submodule update
  $ mvn clean package
