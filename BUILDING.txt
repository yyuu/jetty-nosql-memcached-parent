Building jetty-nosql-memcached-parent
--------------

All relating components have been git-submoduled of this project.
You can build with following procedures.

    $ git clone git://github.com/yyuu/jetty-nosql-memcached-parent.git
    $ cd jetty-nosql-memcached-parent
    $ git submodule init
    $ git submodule update
    $ mvn clean package
